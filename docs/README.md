## 💡 使用场景

### 场景 1：批量下载某个作者的所有视频
1. 打开软件，访问视频号网页版
2. 进入作者主页
3. 滚动加载更多视频
4. 点击工具栏下载图标
5. 选择视频，批量下载

### 场景 2：下载 Home 页面分类视频
1. 切换到想要的分类（如"美食"）
2. 滚动加载更多视频
3. 点击工具栏下载图标
4. 选择视频，批量下载
